# TT-P6-Juegos
Proyecto 6 del bootcamp analista de datos de TripleTen

Felipe Rodríguez Flores

Usando un dataset que contiene información sobre videojuegos, se compara el éxito que cada consola tiene en cada época y cada región.
Se utilizan diversos criterios como la preferencia por géneros en cada región o la influencia de las reseñas.
